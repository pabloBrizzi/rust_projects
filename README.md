# Rust Learning Journey 🦀

This repository documents my Rust learning journey through hands-on exercises, Rustlings challenges, and practical examples inspired by *The Rust Programming Language*.

The goal is not only to learn the Rust language itself, but also to strengthen my understanding of software engineering fundamentals such as memory management, ownership, data structures, algorithmic complexity, and system design concepts.

## Objectives

- Learn Rust fundamentals and idiomatic programming.
- Understand Ownership, Borrowing, and Lifetimes.
- Practice problem-solving through small projects and coding exercises.
- Explore data structures and performance considerations.
- Build a strong foundation for systems programming and high-performance software.
- Apply software engineering concepts through real-world inspired examples.

## Topics Covered

Some of the concepts explored throughout this repository:

- Ownership and Move Semantics
- Borrowing (`&T`, `&mut T`)
- Pattern Matching
- Structs and Methods (`impl`)
- Enums as Event Models
- HashMaps and Frequency Counting
- Unit Testing with `cargo test`
- Basic Algorithmic Complexity (`O(1)`, `O(n)`, `O(n²)`)
- Memory Management (Stack vs Heap)
